# Maintenance Reports

Aplicación web móvil para gestionar reportes de mantenimiento aeronáutico, desde la creación de órdenes hasta la firma y el intercambio de reportes.

## ✈️ Objetivo

Centralizar el proceso de mantenimiento en aviación para que el equipo técnico pueda:

- Crear órdenes de mantenimiento.
- Dar seguimiento a cada orden.
- Consultar el estado actual de la aeronave.
- Visualizar horas disponibles y uso.
- Registrar actualizaciones y pendientes.
- Generar reportes de mantenimiento.
- Firmar reportes.
- Compartir reportes con las áreas involucradas.

## 🧩 Funcionalidades clave

1. **Gestión de órdenes de mantenimiento**
   - Alta de órdenes.
   - Edición y actualización de estado.
   - Historial de seguimiento.

2. **Seguimiento operativo**
   - Estado de aeronaves en tiempo real operativo.
   - Registro de pendientes y actividades en curso.
   - Control de horas disponibles.

3. **Reportería y documentación**
   - Creación de reportes técnicos.
   - Firma de reportes.
   - Compartición de reportes para revisión y control.

## 👥 Usuarios objetivo

- Técnicos de mantenimiento.
- Supervisores de taller/hangar.
- Coordinación de operaciones y mantenimiento.
